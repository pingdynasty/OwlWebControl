# OwlWebControl
Proof-of-concept Web MIDI HTML applications for controlling the OWL family of devices, including
 * OWL Pedal
 * OWL Modular
 * Alchemist
 * Wizard
 * Magus
 * Lich
 * BioSignals
 * OWL Rack
 * Noctua

For more information see [www.rebeltech.org](https://www.rebeltech.org).
